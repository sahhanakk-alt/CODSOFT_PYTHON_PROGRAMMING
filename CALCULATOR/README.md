# 🧮 Calculator

A simple and interactive Calculator application developed using Python. This project performs basic arithmetic operations through a command-line interface and helps beginners understand the fundamentals of Python programming, including functions, conditional statements, and user input handling.

## ✨ Features

* Perform Addition
* Perform Subtraction
* Perform Multiplication
* Perform Division
* Simple and user-friendly interface
* Handles multiple calculations
* Easy to understand and modify

## 🛠️ Technologies Used

* **Python 3**
* Command Line Interface (CLI)
* Python Functions
* Conditional Statements (`if`, `elif`, `else`)
* User Input Handling
* Arithmetic Operators

## 🚀 Getting Started

### Prerequisites

* Python 3.x installed on your system

### Installation

1. Clone this repository:

   ```bash
   git clone https://github.com/your-username/calculator.git
   ```

2. Navigate to the project directory:

   ```bash
   cd calculator
   ```

3. Run the program:

   ```bash
   python calculator.py
   ```

## 📖 Usage

1. Choose an arithmetic operation.
2. Enter two numbers.
3. View the calculated result.
4. Repeat the process or exit the application.
