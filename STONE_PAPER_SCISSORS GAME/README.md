# ✂️ Stone Paper Scissors

A fun and interactive Stone Paper Scissors game developed using Python. In this command-line game, the user competes against the computer, which makes random choices. The project demonstrates Python fundamentals such as conditional statements, loops, functions, and random number generation.

## ✨ Features

* Play against the computer
* Random computer choices
* Interactive command-line interface
* Displays game results instantly
* Option to play multiple rounds
* Easy-to-understand gameplay

## 🛠️ Technologies Used

* **Python 3**
* Command Line Interface (CLI)
* `random` module
* Functions
* Conditional Statements (`if`, `elif`, `else`)
* Loops
* User Input Handling

## 🚀 Getting Started

### Prerequisites

* Python 3.x installed on your system

### Installation

1. Clone this repository:

   ```bash
   git clone https://github.com/your-username/stone-paper-scissors.git
   ```

2. Navigate to the project directory:

   ```bash
   cd stone-paper-scissors
   ```

3. Run the program:

   ```bash
   python stone_paper_scissors.py
   ```

## 📖 Usage

1. Run the application.
2. Choose Stone, Paper, or Scissors.
3. The computer makes a random choice.
4. The winner is determined based on the game rules.
5. Continue playing multiple rounds or exit.
